$(\mathbb{Z}, +)$

عالم الأعداد الصحيحة - حجر الأساس لنظرية الزمر

اولاً البنية الجبرية للزمرة

1. لماذا $\mathbb{Z}$ زمرة؟ (شروط الزمرة)

لتكون مجموعة الأعداد الصحيحة $\mathbb{Z} = \{..., -2, -1, 0, 1, 2, ...\}$ زمرة تحت عملية الجمع، يجب أن تحقق البديهيات الأربع التالية:

  • الانغلاق (Closure): لكل $a, b \in \mathbb{Z}$، فإن حاصل جمعهما يظل عدداً صحيحاً ($a + b \in \mathbb{Z}$). لا يمكن أن يعطينا الجمع عدداً كسرياً.
  • التجميع (Associativity): ترتيب الأقواس لا يهم. لكل $a, b, c \in \mathbb{Z}$، فإن $(a + b) + c = a + (b + c)$.
  • العنصر المحايد (Identity Element): يوجد عنصر فريد وهو الصفر $0$، بحيث لا يؤثر على عملية الجمع: $a + 0 = 0 + a = a$.
  • النظير (Inverse): لكل عنصر $a \in \mathbb{Z}$، يوجد عنصر آخر وهو $-a$ (يسمى النظير الجمعي) بحيث يلغي أثره ليعطي المحايد: $a + (-a) = 0$.

ملاحظة: بما أن $a + b = b + a$، فإنها تسمى زمرة إبدالية (Abelian Group).

قاعدة المعكوس في $\mathbb{Z}$:
معكوس أي عدد $a$ بالنسبة للجمع هو بكل بساطة سالبه $-a$.
مثال: معكوس $17$ هو $-17$ لأن $17 + (-17) = 0$.

2. المولدات (Generators) والزمر الجزئية(Subgroups)

الزمرة الدائرية هي الزمرة التي يمكن بناء جميع عناصرها باستخدام عنصر واحد فقط عبر تكرار العملية (الجمع في حالتنا).

المولدات

يمكننا توليد الزمرة بأكملها باستخدام الرقم $1$ (عن طريق جمعه مع نفسه أو طرحه مراراً وتكراراً). لذلك نسميها زمرة دائرية (Cyclic Group).

أي زمرة جزئية بداخلها تأخذ شكل مضاعفات رقم معين $n$ ونرمز لها بـ $n\mathbb{Z}$. إليك أشهر الأمثلة:

  • الزمرة التافهة: $\langle 0 \rangle = \{0\}$. (الزمرة الجزئية الوحيدة المنتهية هنا!).
  • زمرة الأعداد الزوجية $2\mathbb{Z}$: $\langle 2 \rangle = \{ ..., -4, -2, 0, 2, 4, ... \}$.
  • زمرة مضاعفات الثلاثة $3\mathbb{Z}$: $\langle 3 \rangle = \{ ..., -6, -3, 0, 3, 6, ... \}$.
تمتلك الزمرة $(\mathbb{Z}, +)$ مولدين اثنين فقط، هما:
العدد 1: لأن $1+1=2$، $1+1+1=3$... ونعكسه للحصول على السوالب.
العدد -1: بنفس الطريقة.
ونعبر عن ذلك رياضياً: $$ \mathbb{Z} = \langle 1 \rangle = \langle -1 \rangle $$

3. الزمر الجزئية (Subgroups $n\mathbb{Z}$)

كل زمرة جزئية من $\mathbb{Z}$ تأخذ شكلاً واحداً فقط: مضاعفات عدد صحيح $n$. ونرمز لها بالرمز $n\mathbb{Z}$ أو $\langle n \rangle$.

  • الزمرة التافهة $0\mathbb{Z}$: تحتوي على $\{0\}$ فقط. (وهي الزمرة الجزئية الوحيدة المنتهية).
  • الأعداد الزوجية $2\mathbb{Z}$: $\langle 2 \rangle = \{ ..., -4, -2, 0, 2, 4, ... \}$.
  • مضاعفات الثلاثة $3\mathbb{Z}$: $\langle 3 \rangle = \{ ..., -6, -3, 0, 3, 6, ... \}$.

4. هندسة الزمر الجزئية (العمليات والاحتواء)

هنا يبدأ الجبر المجرد الحقيقي! كيف تتفاعل هذه الزمر الجزئية مع بعضها البعض؟

أ. شرط الاحتواء (Inclusion): متى تكون زمرة جزئية محتواة بالكامل داخل زمرة أخرى؟

القاعدة الذهبية: $$ m\mathbb{Z} \subseteq n\mathbb{Z} \iff n \text{ divides } m $$ مثال: $6\mathbb{Z} \subseteq 2\mathbb{Z}$ لأن مضاعفات الـ 6 (مثل 6، 12، 18) هي تلقائياً مضاعفات للـ 2. لكن العكس غير صحيح!

ب. تقاطع الزمر الجزئية (Intersection): تقاطع أي زمرتين جزئيتين يعطينا زمرة جزئية جديدة تتولد بواسطة المضاعف المشترك الأصغر (LCM) للعددين.

$$ a\mathbb{Z} \cap b\mathbb{Z} = \text{lcm}(a, b)\mathbb{Z} $$ مثال: الأعداد التي تنتمي لـ $4\mathbb{Z}$ و $6\mathbb{Z}$ في نفس الوقت هي مضاعفات الـ 12. إذن: $4\mathbb{Z} \cap 6\mathbb{Z} = 12\mathbb{Z}$.

جـ. جمع الزمر الجزئية (Addition of Subgroups): إذا جمعنا كل عنصر من $a\mathbb{Z}$ مع كل عنصر من $b\mathbb{Z}$، سنحصل على زمرة جزئية تتولد بواسطة القاسم المشترك الأكبر (GCD).

$$ a\mathbb{Z} + b\mathbb{Z} = \text{gcd}(a, b)\mathbb{Z} $$ مثال: $4\mathbb{Z} + 6\mathbb{Z} = 2\mathbb{Z}$. (تُعرف هذه بخاصية بيزوت Bezout's Identity).

5. الدليل (Index of a Subgroup)

دليل الزمرة الجزئية $n\mathbb{Z}$ داخل $\mathbb{Z}$، والذي نرمز له بالرمز $[\mathbb{Z} : n\mathbb{Z}]$، يمثل عدد "المصاحبات" (Cosets) المختلفة التي تصنعها هذه الزمرة.

بشكل مبسط، الدليل هو ببساطة العدد $n$.
مثال: $[\mathbb{Z} : 3\mathbb{Z}] = 3$. والمصاحبات الثلاثة هي: (مضاعفات 3)، (مضاعفات 3 مضافاً إليها 1)، (مضاعفات 3 مضافاً إليها 2).

المقارنة : اللانهائي $\mathbb{Z}$ ضد المنتهي $Z_n$

لقد درست زمرة التطابق (المعيارية) $Z_n$ في الفصول السابقة، فكيف تختلف عن زمرتنا اليوم $\mathbb{Z}$؟

وجه المقارنة الزمرة اللانهائية $(\mathbb{Z}, +)$ الزمرة المنتهية $(Z_n, +_n)$
رتبة الزمرة (الحجم) لانهائية ($\infty$) منتهية ($n$ عنصر)
المولدات (Generators) يوجد مولدان فقط: $1$ و $-1$ أي رقم $k$ يكون $\gcd(k, n) = 1$
معكوس العدد $a$ $-a$ (نفس الرقم بإشارة سالبة) $n - a$ (حسب مود الزمرة)
الزمر الجزئية عدد لا نهائي ($n\mathbb{Z}$) محدودة (قواسم العدد $n$ فقط)
ثانياً مستكشف الزمر الجزئية

أدخل أي رقم $n$ لترى الزمرة الجزئية $n\mathbb{Z}$ على خط الأعداد.
